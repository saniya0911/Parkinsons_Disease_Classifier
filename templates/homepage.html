Parkinson's Disease Prediction

Parkinson’s Disease is a progressive disorder of the nervous system that affects movement. This application aims to assist early detection of Parkinson’s Disease using machine learning models based on vocal features or other clinical data.

By analyzing the input parameters, this system can help provide insights into whether a patient might be affected by Parkinson’s Disease, which can assist in early diagnosis and treatment.
